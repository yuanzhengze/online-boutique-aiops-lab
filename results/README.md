# 结果材料目录

本目录用于统一保存截图、指标数据、测试结果和实验图表。

## 子目录

- `screenshots/`：部署、监控、测试、Agent 邮件等截图。
- `metrics/`：Prometheus 导出数据、CSV、JSON 等。
- `test-results/`：Selenium、JMeter 测试结果。
- `figures/`：论文复现和算法对比图表。

## 命名建议

```text
日期-模块-内容描述.扩展名
```

示例：

```text
0601-online-boutique-running.png
0605-jmeter-result.csv
0606-agent-email-result.png
0607-algorithm-comparison.png
```
