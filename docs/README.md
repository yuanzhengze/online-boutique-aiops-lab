# 文档索引

本目录用于存放项目协作文档、模板、报告材料和展示材料。每位成员提交内容前，先查看对应模板。

## 必读文档

- `../README.md`：项目入口说明。
- `../plan.md`：10 天开发计划。
- `../项目协作文档.md`：成员协作规则。
- `../agent.md`：Agent 协作指南。

## 本目录文档

- `deployment.md`：部署记录模板。
- `daily-progress.md`：每日进度记录。
- `experiment-record-template.md`：实验记录模板。
- `member-contribution.md`：成员贡献记录。
- `report-outline.md`：报告大纲。
- `ppt-outline.md`：PPT 大纲。
- `module-checklist.md`：各模块交付检查清单。

## 提交材料要求

每位成员每天至少提交一种阶段性材料：

- 代码或脚本。
- 配置文件。
- 运行截图。
- 实验结果。
- 日志片段。
- 报告段落。
- PPT 页面草稿。

所有截图统一放入 `results/screenshots/`，实验结果放入 `results/metrics/`、`results/test-results/` 或 `results/figures/`。
